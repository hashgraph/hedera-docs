description: "TopicID protobuf type uniquely identifying a Hedera Consensus Service topic by shard, realm, and topic number used to submit and query messages."
---


Unique identifier for a topic (used by the consensus service)

| Field      | Description                                           |
| ---------- | ----------------------------------------------------- |
| `shardNum` | The shard number (nonnegative)                        |
| `realmNum` | The realm number (nonnegative)                        |
| `topicNum` | Unique topic identifier within a realm (nonnegative). |
