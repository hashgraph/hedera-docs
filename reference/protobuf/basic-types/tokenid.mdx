---
title: "TokenID"
description: "Hedera protobuf reference for the TokenID message, a unique identifier for a Hashgraph token built from shard, realm, and token number components."
---


A unique identifier for a token.

| Field      | Description                |
| ---------- | -------------------------- |
| `shardNum` | A nonnegative shard number |
| `realmNum` | A nonnegative realm number |
| `tokenNum` | A nonnegative token number |
