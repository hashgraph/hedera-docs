---
title: "FileID"
description: "FileID is the Hedera protobuf identifier for a stored file, expressed as a shard, realm, and file number triplet used by the file service."
---


The ID for a file

| Field      | Description                                       |
| ---------- | ------------------------------------------------- |
| `shardNum` | The shard number (nonnegative)                    |
| `realmNum` | The realm number (nonnegative)                    |
| `fileNum`  | A nonnegative File number unique within its realm |
