---
title: "TokenID"
description: "TokenID protobuf type uniquely identifying a Hedera Token Service fungible or non-fungible token by shard, realm, and token number."
---


A unique identifier for a token.

| Field      | Description                |
| ---------- | -------------------------- |
| `shardNum` | A nonnegative shard number |
| `realmNum` | A nonnegative realm number |
| `tokenNum` | A nonnegative token number |
