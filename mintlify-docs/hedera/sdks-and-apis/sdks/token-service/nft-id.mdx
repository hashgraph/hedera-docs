---
title: "NFT ID"
---


The ID of a non-fungible token (NFT). The NFT ID is composed of the [**token ID**](/hedera/sdks-and-apis/sdks/token-service/token-id) and a **serial number**.

| Constructor                     | Description                  |
| ------------------------------- | ---------------------------- |
| `new NftId(<tokenId>,<serial>)` | Initializes the NftId object |

```java
new NftId()
```

## Methods

| Method                   | Type      | Requirement |
| ------------------------ | --------- | ----------- |
| `NftId.fromString(<id>)` | String    | Optional    |
| `NftId.fromBytes(<id>)`  | bytes \[] | Optional    |

<CodeGroup>

```java Java
new NftId(new TokenId(0,0,2), 56562);

// v2.0.11
```


```javascript JavaScript
new NftId(new TokenId(0,0,2), 56562);

// v2.0.28 
```


```java Go
nftId := hedera.NftID{
    TokenID: tokenId,
		SerialNumber: serialNum,
}

// v2.1.13
```

</CodeGroup>
