---
title: "Setting"
description: "Setting protobuf type representing a single name, value, and data entry inside a Hedera ServicesConfigurationList describing node configuration properties."
---


| Field   | Description                       |
| ------- | --------------------------------- |
| `name`  | name of the property              |
| `value` | value of the property             |
| `data`  | any data associated with property |
