---
title: "FileID"
description: "Hedera protobuf reference for the FileID message, identifying a file stored on the Hashgraph network using shard, realm, and file number values."
---


The ID for a file

| Field      | Description                                       |
| ---------- | ------------------------------------------------- |
| `shardNum` | The shard number (nonnegative)                    |
| `realmNum` | The realm number (nonnegative)                    |
| `fileNum`  | A nonnegative File number unique within its realm |
