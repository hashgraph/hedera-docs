---
title: "Setting"
description: "Hedera protobuf reference for the Setting message, a single name-value-data property entry used inside Hashgraph services configuration lists."
---


| Field   | Description                       |
| ------- | --------------------------------- |
| `name`  | name of the property              |
| `value` | value of the property             |
| `data`  | any data associated with property |
