---
title: "TokenKycStatus"
description: "TokenKycStatus protobuf enum listing possible KYC statuses returned for an account in TokenGetInfoQuery and CryptoGetInfoResponse responses."
---


Possible KYC statuses returned on TokenGetInfoQuery or CryptoGetInfoResponse in TokenRelationship.

| Enum Name          | Description     |
| ------------------ | --------------- |
| `KycNotApplicable` | KYC n/a         |
| `Granted`          | KYC was granted |
| `Revoked`          | KYC was revoked |
