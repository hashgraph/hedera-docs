## Hedera Network Services

Learn about the suite of network services offered on Hedera and how each can be used to build decentralized applications.

<CardGroup cols={3}>
<Card title="Smart Contracts" href="/hedera/sdks-and-apis/hedera-api/smart-contracts" img="/images/index/index-13.png">
An EVM-based smart contract service. Build and deploy contracts using Solidity or port over an existing smart contract with ease.
</Card>

<Card title="Token Service" href="/hedera/sdks-and-apis/hedera-api/token-service" img="/images/index/index-14.png">
Configure, mint, and manage native fungible and non-fungible tokens or using ERC-20 & 721 standards on the Hedera network.
</Card>

<Card title="Consensus Service" href="/hedera/sdks-and-apis/sdks/consensus-service" img="/images/index/index-15.png">
Incorporate scalable, real-time, auditable logs of events with consensus timestamps into any web2 or permissionless web3 application.
</Card>

<Card title="File Service" href="/hedera/sdks-and-apis/sdks/file-service" img="/images/index/index-16.png">

Hedera File Service is a decentralized file storage platform that allows users to securely store and access files on a distributed network of computers using hash as a file identifier.

</Card>
</CardGroup>

## EVM-Compatible Tools

Continue utilizing familiar Ethereum development tools such as Web3.js, Truffle, Ethers, Hardhat, and Foundry to build on Hedera using the [JSON-RPC Relay](/hedera/core-concepts/smart-contracts/json-rpc-relay). As an Ethereum developer, your workflow does not have to change.

<CardGroup cols={3}>
<Card title="Create an ECDSA Account" href="https://github.com/hashgraph/hedera-json-rpc-relay/blob/main/docs/examples/account-alias-ecdsa.js" img="/images/index/index-17.png" >
Create an ECDSA account that works with JSON-RPC to use EVM-compatible tooling, libraries, and wallets.
</Card>

<Card title="Set Up Metamask" href="https://github.com/hashgraph/hedera-json-rpc-relay/blob/main/docs/transfer-guide.md" img="/images/index/index-18.png">
Add a Hedera network to Metamask using the [Hashio](http://hashio.io/) implementation of the JSON-RPC Relay. Sign transactions and queries to test your Hedera-powered application.
</Card>

<Card title="Libraries & Tooling" href="https://github.com/hashgraph/hedera-json-rpc-relay/tree/main/tools" img="/images/index/index-19.png">
Integrate Hedera into Ethers, web3.js, Truffle, or Hardhat for building, debugging, and deploying your smart contract applications on the Hedera network.
</Card>

<Card title="JSON-RPC Relay" href="/hedera/core-concepts/smart-contracts/json-rpc-relay" img="/images/index/index-20.png">
JSON-RPC makes it easy for existing EVM-based libraries, dev tools, and wallets to work effortlessly on Hedera.
</Card>
</CardGroup>

## Developer Tools and Integrations

Explore various developer tools and resources on Hedera that help make building and maintaining your decentralized application even easier.

<CardGroup cols={3}>
<Card title="Network Explorers" href="/hedera/networks/community-mirror-nodes" img="/images/index/index-21.png">
Visualizations & APIs for historical network data, including account & token details, transactions, tokens, contracts, topics, and schedules.
</Card>

<Card title="Wallet Integrations" href="/hedera/sdks-and-apis/sdks#ecosystem-wallet-support" img="/images/index/index-22.png">
Onboard users into your application with ease. Wallets on Hedera offer transaction signing, management of HBAR, NFTs, and fungible tokens.
