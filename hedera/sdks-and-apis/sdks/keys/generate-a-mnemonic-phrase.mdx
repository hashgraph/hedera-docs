title: "Generate a mnemonic phrase"
---


Generate a 12 or 24-word mnemonic phrase that can be used to recover the private keys that are associated with it.

<table><thead><tr><th></th><th></th><th></th></tr></thead><tbody><tr><td><strong>Method</strong></td><td><strong>Type</strong></td><td><strong>Description</strong></td></tr><tr><td><code>Mnemonic.generate24()</code></td><td>Mnemonic</td><td>Generates a 24-word recovery phrase that can be used to recover a private key</td></tr><tr><td><code>Mnemonic.generate12()</code></td><td>Mnemonic</td><td>Generates a 12-word recovery phrase that can be used to recover a private key</td></tr></tbody></table>

<CodeGroup>

```java Java
// 24-word recovery phrase
Mnemonic mnemonic = Mnemonic.generate24();
System.out.println("mnemonic 24 word = " + mnemonic);


//12 word recovery phrase
Mnemonic mnemonic12 = Mnemonic.generate12();
System.out.println("mnemonic 12 word = " + mnemonic12);

//v2.0.0
```


