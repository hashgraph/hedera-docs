## Overview

Agent Lab is an interactive development environment within the [Hedera Developer Portal](https://portal.hedera.com) for visually designing, building, and testing AI agents powered by the [Hedera Agent Kit](https://docs.hedera.com/hedera/open-source-solutions/ai-studio-on-hedera/hedera-agent-kit). It is accessible on the [**Hedera Portal under the Agent Lab tab**](https://portal.hedera.com/agent-lab).
 
The tool provides a guided path from no-code experimentation to production-ready agent code — supporting both non-technical users who want to prototype quickly and developers who want a working starting point they can customize and deploy.
 
There are two AI models at work in Agent Lab:
 
- The **AI coding assistant** (right panel in the code editor) helps you write and extend Hedera Agent Kit code.
- The **agent you build** is the Hedera AI agent that executes on-chain transactions when you interact with it in the Run view.
