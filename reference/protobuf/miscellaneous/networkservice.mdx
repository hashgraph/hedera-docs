The requests and responses for different network services.

#### NetworkService

| RPC              | Request | Response | Comments                                                       |
| ---------------- | ------- | -------- | -------------------------------------------------------------- |
| `getVersionInfo` | Query   | Response | Retrieves the active versions of Consensus Node and HAPI proto |
